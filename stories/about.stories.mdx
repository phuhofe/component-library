# Adstate component library

## About the library

**Why does this UI Library exist?**

To unify and maintain consistency of the most commonly used UI elements at Adstate and to provide a platform for collaborative constant improvement. It's a part of the whole Design System.

This part of the Design System is dedicated to **application development**.
