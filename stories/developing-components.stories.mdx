# How to develop components

1. We are following the atomic web design structure. You should familiarize yourself with the concept.
2. Ideally the design of the component exsists in Figma before you start development.
3. `git clone git@gitlab.com:adstate_as/adstate-design-system/component-library.git`.
4. `npm install`
5. `npm run ng g c NameOfYourComponent`
6. Discard the auto generated changes in `/projects/flora/src/lib/flora.module.ts`
7. Create a module file: `/projects/flora/src/lib/name-of-your-component/name-of-your-component.module.ts`
   Add the following content:

```ts
import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { NameOfYourComponentComponent } from './name-of-your-component.component';
import { CardModule } from '../card/card.module';

// CardModule is just added as example here, showing you how to use other components inside yours,
// remove it if you don't need it
@NgModule({
  declarations: [NameOfYourComponentComponent],
  imports: [CommonModule, CardModule],
  exports: [NameOfYourComponentComponent],
})
export class NameOfYourComponentModule {}
```

8. Add your module to: `/projects/flora/src/public-api.ts`, like so: `export { NameOfYourComponentModule } from './lib/name-of-your-component/name-of-your-component.module';`
9. Add a `name-of-your-component.stories.mdx` in `/stories/TYPE/name-of-your-component.stories.ts`.

Feel free to look at the "Molecules / Passing Card" component for a good example.
